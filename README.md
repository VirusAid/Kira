## ✨ Возможности

| | |
|---|---|
| 🧬 **Kira Core** | Собственная платформа управления ПК: **Intent Parser → Command Engine → Action API → Controllers**. Команды выполняются локально и мгновенно, **без обращения к LLM** |
| 🔮 **Понимание смысла** | Гибридный разбор намерений: сначала точные шаблоны, затем **семантический слой на эмбеддингах** — команда ловится в любой формулировке, а не только заученной |
| 🤖 **Агент** | Встроен в ядро: ставишь цель — Kira сама строит план и выполняет его до конца, проверяя каждый шаг. Долгие задачи уносит в **фон** (диалог не блокируется), а узкие подзадачи поручает фокусному **суб-агенту** |
| 🗣 **Голос** | Непрерывное прослушивание, распознавание речи, слово-активатор «Кира», перебивание на полуслове, естественный ответ голосом |
| 🎭 **Личности** | 7 готовых характеров в один клик — от заботливой Киры до делового ассистента и дворецкого — плюс свободная настройка |
| 🧠 **Живая память** | Собственный характер, эмоции, долговременная память и персональный профиль, который растёт и учится у тебя со временем |
| 🖥 **Управление ПК** | Программы, файлы, громкость, яркость, окна, мышь и клавиатура, питание, процессы — через надёжный протокол действий с **отменой** |
| 👁 **Зрение** | Смотрит на экран, видит **все запущенные процессы**, анализирует и действует по увиденному, самоисправляется после действий |
| ⚡ **Навыки (Abilities)** | Опиши умение обычным языком — Kira запомнит и будет выполнять его сама. Навыки можно передавать файлом |
| 🎯 **AI Actions** | Горячая клавиша по выделенному тексту и пункт в меню Проводника: перевести, объяснить, переписать, сократить |
| 🌅 **Проактивность** | Утренний брифинг, умные напоминания, слежение за системой, заботливые подсказки |
| 🧩 **Автоматизация** | Протоколы — последовательности действий одним кликом, запуск по расписанию или событию |
| 🌐 **Интеграции** | Obsidian, Notion, Google Calendar, Gmail, Discord, Telegram (бот + личный аккаунт) |
| 🎨 **Интерфейс** | Тёмный футуристичный дизайн, анимированная эмблема-мандала, оверлей поверх всех окон |

## ⚡ Команды — 70 локальных действий

Всё ниже выполняется **ядром напрямую**, без обращения к нейросети:

| Область | Примеры команд |
|---------|----------------|
| **Приложения** | браузер, калькулятор, блокнот, проводник, терминал, Paint, VS Code, Docker, любое приложение по имени |
| **Медиа** | плей/пауза, следующий/предыдущий трек, громче/тише, точная громкость, мьют, музыка и видео по названию |
| **Система** | скриншот, яркость, загрузка CPU/RAM, место на диске, заряд батареи, IP-адрес, процессы, параметры, диспетчер |
| **Окна** | свернуть всё, свернуть / развернуть / закрыть активное окно |
| **Правка** | копировать, вставить, вырезать, выделить всё, сохранить |
| **Файлы** | открыть и создать папку, открыть файл, копировать / переместить / переименовать / удалить |
| **Веб** | открыть сайт, «загугли…», поиск на YouTube, погода |
| **Время** | который час, какое сегодня число, таймер |
| **Утилиты** | конвертер единиц, курс валют и крипты, QR-код, ИМТ, скорость интернета |
| **Питание** | блокировка, сон, гибернация, выключение, перезагрузка |
| **Буфер и корзина** | читать / писать буфер, открыть и очистить корзину |
| **Отмена** | «отмени» — откат последнего действия |

Любую из них можно сказать в произвольной форме — сработает [семантический слой](#-архитектура--local-first).
Всё, что не команда (письма, объяснения, сложные цели) — уходит в AI Router и агента.

## 🏗 Архитектура · *Local First*

Kira — **не обёртка над языковой моделью**. Каждый запрос сначала пытается решить
собственное ядро — быстро, офлайн и бесплатно. LLM — лишь один из модулей, и запрос
уходит к нему только когда ядро честно понимает, что задача не для него.

```
   Пользователь · чат / голос / hotkey / автоматизация / агент
                              │
                              ▼
        ┌─────────────────────────────────────────────┐
        │              INTENT PARSER                   │
        │   1. Точные шаблоны (regex)  — мгновенно      │
        │   2. Семантика (эмбеддинги)  — по смыслу      │
        └─────────────────────────────────────────────┘
                              │  распознано локально
                              ▼
        ┌─────────────────────────────────────────────┐
        │             COMMAND ENGINE                   │
        │   валидация · подтверждение опасного ·        │
        │   исполнение · история · отмена (undo)        │
        └─────────────────────────────────────────────┘
                              │
                              ▼
        ┌─────────────────────────────────────────────┐
        │       ACTION API  ·  70 действий             │
        │   единый контракт: execute / undo / validate  │
        └─────────────────────────────────────────────┘
                              │
                              ▼
     CONTROLLERS · Browser · Application · Media · File · Power · Window ·
              Input · Search · Clipboard · System · Notification · Git
                              │
                              ▼
                          Windows

                   ↘  (не распознано локально)
       AI Router → LLM-конвейер → те же Actions через протокол [[kira:…]]
```

**Семантический слой** сравнивает фразу с эмбеддингами команд (модель
`paraphrase-multilingual-MiniLM`, локально, офлайн). Порог подобран эмпирически ради
**высокой точности**: лучше честно отдать запрос нейросети, чем выполнить не то.
Так «сделай погромче», «приглуши музыку» или «что там с погодой» срабатывают локально,
даже если точного шаблона под них нет.

Каждое действие автоматически попадает в **историю, шину событий и журнал** и одинаково
доступно из чата, голоса, автоматизаций, агентов и LLM-протокола. Новый Action — это
один объект в реестре, без единой правки в движке *(Open/Closed, Clean Architecture, DI)*.

```
src/
├── shared/            Общие типы (контракт main ↔ renderer), пресеты личностей
├── main/
│   ├── core/          ★ KIRA CORE — сердце платформы
│   │   ├── intent.ts          Intent Parser (regex, чистый и тестируемый)
│   │   ├── semanticIntent.ts  Семантический слой (эмбеддинги)
│   │   ├── engine.ts          Command Engine
│   │   ├── registry.ts        Реестр команд (id, алиасы, шаблоны, права)
│   │   ├── actions/           Action API — все возможности единообразно
│   │   ├── controllers/       Доменные контроллеры
│   │   ├── bus.ts             Шина событий ядра
│   │   └── history.ts         История действий и отмена
│   └── modules/
│       ├── ai/        AI Router: LLM-конвейер, личность, память, голос, зрение
│       ├── system.ts  Низкоуровневый драйвер Windows (экран, процессы, ввод)
│       ├── abilities.ts · automation.ts · protocols.ts · pulse.ts …
│       └── ipc.ts     Все IPC-каналы
├── preload/           Безопасный мост window.kira
└── renderer/          React-интерфейс (Zustand, голосовой режим, экраны)

tests/core.test.ts     Тесты ядра: npm run test:core  (47 проверок)
```

Все данные хранятся локально в `%APPDATA%/kira`. В сеть уходят только запросы
к выбранному провайдеру ИИ.

## 🤖 Модель ИИ

Kira работает с пятью провайдерами — у каждого есть бесплатный доступ.
Ключ вставляется в **Настройки → Модели ИИ** (список моделей подтягивается вживую).

| Провайдер | Стоимость | Где взять ключ |
|-----------|-----------|----------------|
| **Groq** *(рекомендуется)* | Бесплатно, очень быстро + бесплатный Whisper для голоса | https://console.groq.com/keys |
| **OpenRouter** | Модели с суффиксом `:free` бесплатны | https://openrouter.ai/keys |
| **Google Gemini** | Щедрый бесплатный tier, отличное «зрение» | https://aistudio.google.com/apikey |
| **DeepSeek** | Мощные модели V3 и R1 (рассуждения), недорого | https://platform.deepseek.com/api_keys |
| **Ollama** | Полностью локально и офлайн | https://ollama.com |

## 🎙 Голос Kira

Три движка синтеза с автоматическим откатом **Silero → Edge → системный**:

1. **Silero** *(по умолчанию)* — локальный нейросетевой голос, лучшее качество русского
   среди бесплатных, работает офлайн на CPU. Голоса: xenia, kseniya, baya, aidar, eugene.
2. **Edge** — облачная нейросеть Microsoft, голос «Светлана». Нужен интернет, без ключа.
3. **Системный** — голоса Windows, всегда доступны офлайн.

Голосовой ввод — бесплатное распознавание Whisper (через Groq) с офлайн-резервом Vosk.
Слово-активатор «Кира» работает полностью локально.
