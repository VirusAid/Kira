# ⭐ Kira

### Персональный ИИ-ассистент нового поколения для Windows

Живой голосовой помощник с характером: понимает речь, видит экран,
управляет компьютером, помнит твою жизнь и учится новому — всё в едином
футуристичном интерфейсе.

## ✨ Возможности

| | |
|---|---|
| 🤖 **Агент** | Встроен в ядро: ставишь цель — Kira сама строит план и выполняет его до конца своими инструментами, проверяя каждый шаг |
| 🗣 **Голос** | Непрерывное прослушивание, распознавание речи, слово-активатор «Кира», перебивание, естественный ответ голосом |
| 🧠 **Живая личность** | Собственный характер, эмоции, долговременная память и персональный профиль, который растёт и учится у тебя со временем |
| 🖥 **Управление ПК** | Программы, файлы, громкость, яркость, окна, мышь и клавиатура, питание — через надёжный протокол действий |
| 👁 **Зрение** | Смотрит на экран, анализирует и действует по увиденному, самоисправляется после действий |
| ⚡ **Навыки (Abilities)** | Опиши умение обычным языком — Kira запомнит и будет выполнять его сама. Навыки можно передавать файлом |
| 🎯 **AI Actions** | Горячая клавиша по выделенному тексту и пункт в меню Проводника: перевести, объяснить, переписать, сократить |
| 🔮 **Проактивность** | Утренний брифинг, умные напоминания, слежение за системой, заботливые подсказки |
| 🧩 **Протоколы и автоматизация** | Последовательности действий одним кликом, запуск по расписанию/событию |
| 🌐 **Интеграции** | Obsidian, Notion, Google Calendar, Gmail, Discord, Telegram (бот + личный аккаунт) |
| 🎨 **Интерфейс** | Тёмный футуристичный дизайн, анимированная эмблема, оверлей поверх всех окон |

## 🚀 Быстрый старт

**Автоустановка (проще всего):** дважды кликни **`Установить Kira.bat`** — он поставит
зависимости Node и все локальные Python-подсистемы (голос, память, узнавание голоса,
офлайн-активацию). Нужны предустановленные [Node.js](https://nodejs.org) (LTS) и
[Python 3](https://python.org) (с галочкой «Add python.exe to PATH»).
Затем запускай через **`Запустить Kira.bat`**.

**Вручную:**

```bash
npm install
npm run dev        # запуск в режиме разработки
npm run build      # production-сборка (out/)
npm run dist       # установщик Windows (release/)
```

## 🤖 Модель ИИ

Kira работает с четырьмя провайдерами — у каждого есть бесплатный доступ.
Ключ вставляется в **Настройки → Модели ИИ**.

| Провайдер | Стоимость | Где взять ключ |
|-----------|-----------|----------------|
| **Groq** *(рекомендуется)* | Бесплатно, очень быстро + бесплатный Whisper для голоса | https://console.groq.com/keys |
| **OpenRouter** | Модели с суффиксом `:free` бесплатны | https://openrouter.ai/keys |
| **Google Gemini** | Щедрый бесплатный tier, хорошее «зрение» | https://aistudio.google.com/apikey |
| **DeepSeek** | Мощные модели V3 и R1 (рассуждения), недорого | https://platform.deepseek.com/api_keys |
| **Ollama** | Полностью локально и офлайн | https://ollama.com |

## 🎙 Голос Kira

Три движка синтеза с автоматическим откатом **Silero → Edge → системный**:

1. **Silero** *(по умолчанию)* — локальный нейросетевой голос, лучшее качество русского
   среди бесплатных, работает офлайн на CPU. Голоса: xenia, kseniya, baya, aidar, eugene.
2. **Edge** — облачная нейросеть Microsoft, голос «Светлана». Нужен интернет, без ключа.
3. **Системный** — голоса Windows, всегда доступны офлайн.

Для голосового ввода используется бесплатное распознавание речи Whisper (через Groq).

## 🏗 Архитектура

```
src/
├── shared/            Общие типы (контракт main ↔ renderer)
├── main/              Electron main-процесс (Node.js)
│   └── modules/
│       ├── ai/        Клиент ИИ, личность и память, оркестратор действий
│       ├── system.ts  Управление Windows, зрение, ввод
│       ├── abilities.ts  Навыки (Abilities / Skills)
│       ├── automation.ts, protocols.ts, integrations.ts, pulse.ts …
│       └── ipc.ts     Все IPC-каналы
├── preload/           Безопасный мост window.kira
└── renderer/          React-интерфейс (Zustand, голосовой режим, экраны)
```

Все данные хранятся локально в `%APPDATA%/kira`. В сеть уходят только запросы
к выбранному провайдеру ИИ.
