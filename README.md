# Kira — персональный ИИ-помощник уровня JARVIS

Автономный настольный ассистент для Windows: голосовое общение, компьютерное
зрение, управление системой, автоматизация, долговременная память и проекты —
в едином футуристичном интерфейсе.

## Быстрый старт

**Автоустановка (проще всего):** дважды кликни **`Установить Kira.bat`** —
он поставит зависимости Node и все локальные Python-подсистемы (голос Silero,
семантическую память, узнавание голоса, эмоции, офлайн-активацию). Нужны
предустановленные [Node.js](https://nodejs.org) (LTS) и [Python 3](https://python.org)
(с галочкой «Add python.exe to PATH»). Затем запускай через `Запустить Kira.bat`.

**Вручную:**

```bash
npm install
npm run dev        # запуск в режиме разработки
npm run build      # production-сборка (out/)
npm run dist       # установщик Windows (release/)
```

## Подключение бесплатной модели ИИ

Kira работает с четырьмя провайдерами, у каждого есть бесплатный доступ. Ключ
вставляется в **Настройки → Модели ИИ**.

| Провайдер | Стоимость | Где взять ключ |
|-----------|-----------|----------------|
| **Groq** (рекомендуется) | Бесплатно, очень быстро + бесплатный Whisper для голоса | https://console.groq.com/keys |
| **OpenRouter** | Модели с суффиксом `:free` бесплатны | https://openrouter.ai/keys |
| **Google Gemini** | Щедрый бесплатный tier, хорошее «зрение» | https://aistudio.google.com/apikey |
| **Ollama** | Полностью локально и офлайн | https://ollama.com (`ollama pull llama3.1`) |

Для голосового ввода нужен ключ Groq — он же даёт бесплатное распознавание речи
(Whisper).

**Голос Kira** — три движка синтеза (Настройки → Голос):

1. **Silero (по умолчанию)** — локальный нейросетевой голос, лучшее качество
   русского среди бесплатных, работает офлайн на CPU (не занимает GPU). Голос по
   умолчанию — «Ксения» (`xenia`), можно переключить на baya, kseniya, aidar,
   eugene. Требует Python-пакет `torch` (см. ниже).
2. **Edge (облако)** — нейросеть Microsoft, голос «Светлана». Нужен интернет,
   ключ не требуется.
3. **Системный** — голоса Windows, всегда доступны офлайн.

Kira автоматически откатывается по цепочке Silero → Edge → системный, если
предыдущий недоступен.

### Установка локального голоса Silero

Silero использует Python + PyTorch (CPU). Установка один раз:

```powershell
pip install --target "E:\kira 1.0\pyenv" torch numpy --index-url https://download.pytorch.org/whl/cpu
```

Модель (~40 МБ) скачивается автоматически при первом запуске голоса и кэшируется
в `%APPDATA%\kira\silero`. Если официальный сервер моделей недоступен, Kira
берёт модель с зеркала на HuggingFace.

### Музыка и веб

Скажи Kira «включи <трек/исполнитель>» — она найдёт видео на YouTube и запустит
его с автозапуском. Также доступны быстрый ввод трека и медиа-кнопки в правой
панели, управление любым плеером через мультимедийные клавиши.

## Архитектура

```
src/
├── shared/            Общие типы (контракт main ↔ renderer)
├── main/              Electron main-процесс (Node.js)
│   ├── index.ts       Точка входа, окно, жизненный цикл
│   └── modules/
│       ├── storage.ts     Файловая JSON-БД (атомарная, отложенная запись)
│       ├── db.ts          Коллекции данных
│       ├── settings.ts    Настройки + значения по умолчанию
│       ├── logger.ts      Журнал событий → renderer
│       ├── system.ts      Управление Windows (программы, громкость, яркость,
│       │                  скриншоты, процессы, питание, буфер, ввод)
│       ├── files.ts       Файловый менеджер
│       ├── protocols.ts   Движок последовательностей действий
│       ├── automation.ts  Триггеры (cron, запуск, наблюдение за папкой)
│       ├── search.ts      Глобальный поиск
│       ├── ipc.ts         Регистрация всех IPC-каналов
│       └── ai/
│           ├── client.ts  Единый стриминговый клиент (OpenAI-совместимый)
│           ├── kira.ts    Личность, память, системные инструменты
│           └── chat.ts    Оркестратор: стриминг → действия → продолжение
├── preload/           Безопасный мост window.kira
└── renderer/          React-интерфейс
    └── src/
        ├── state/     Zustand-сторы
        ├── voice/     Голосовой режим (VAD + Whisper + TTS)
        ├── components/ Оболочка, панели, поиск, диалоги
        └── views/     9 экранов
```

## Возможности

- **Чат** — стриминг, Markdown/код/таблицы, редактирование, копирование,
  перегенерация, история, поиск, «зрение» (скриншот в запрос).
- **Управление ПК** — Kira выполняет действия через текстовый протокол
  `[[kira:...]]`, надёжно работающий на любых бесплатных моделях. Опасные
  действия требуют подтверждения.
- **Голос** — непрерывное прослушивание, детекция речи по громкости, перебивание,
  автоответ голосом, анимация активности.
- **Протоколы** — визуальный редактор последовательностей действий; запуск,
  остановка, дублирование, импорт/экспорт.
- **Автоматизация** — по расписанию (cron), при запуске, при появлении файла.
- **Память** — 6 категорий, редактирование через интерфейс, автосохранение фактов.
- **Проекты** — цели, прогресс, заметки, теги, привязанные диалоги.
- **Файлы** — навигация, поиск, быстрый просмотр и редактирование текста.
- **Логи, глобальный поиск (Ctrl+K), большой раздел настроек, бэкап данных.**

Все данные хранятся локально в `%APPDATA%/kira/data`. В сеть уходят только
запросы к выбранному провайдеру ИИ.
